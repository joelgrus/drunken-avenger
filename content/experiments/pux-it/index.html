Pux-It

A Spot It clone (sort of) built using PureScript and Pux.

Each pair of cards have *exactly* one image in common. Click on cards to select or unselect them. If you select two cards, the common image will flash. (You can't select more than two cards.)

For gory details read the blog post and/or check out the repo on GitHub.
